# The Simulation Holds

The website for the book *The Simulation Holds: On Reality, the Self, and
the Fictions That Run Us.*

Live at [the-simulation-holds.com](https://the-simulation-holds.com).

## What this is

A single self-contained HTML page. Scrolling descends through a receding
series of gates that does not bottom out. The page makes no external
network requests: the typeface is subset and embedded, the texture is
generated, the icons are local. No CDN, no analytics, no trackers.

## The book

- Object (paperback): on Amazon
- Text (free): [archive.org/details/the-simulation-holds](https://archive.org/details/the-simulation-holds)

The text is dedicated to the public domain.

## Hosting

Served as a static site on GitHub Pages with a custom apex domain. Any
static host works; drop the files at the web root.
